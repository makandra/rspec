== RSpec 1 for Ruby 2.3

This is a fork of the rspec 1.x repository.

RSpec 1 is unmaintained. We made slight changes to allow rspec 1 run on Ruby 2.3.x.


For current RSpec versions, see http://rspec.info for an overview and any of the following repos for code:

* http://github.com/rspec/rspec
* http://github.com/rspec/rspec-core
* http://github.com/rspec/rspec-mocks
* http://github.com/rspec/rspec-expectations
